## Epics Included

### **Epic 1: Enhanced Authentication & Security**
- **Scope**: OAuth Google + MFA + SSO com intelligent session management
- **Architecture**: Dual-mode authentication + instant rollback capability
- **Key Features**: 
  - Multi-factor authentication (SMS + Email)
  - Role-based permissions (Owner, Manager, Staff, Patient)
  - SSO integration com Google/Microsoft
  - Session management com auto-logout (30min inactivity)
  - LGPD-compliant consent management

  - Automated reminder system (SMS/Email/WhatsApp)
