## 🚀 Como Usar

### 1. Navegação
- Acesse "Contas a Pagar" no menu lateral
- Selecione "Analytics" para ver o dashboard completo

### 2. Gerenciamento de Documentos
- Nos formulários de fornecedores/contas a pagar, use a aba "Documentos"
- Faça upload de arquivos, categorize por tipo
- Visualize, baixe ou exclua documentos existentes

### 3. Performance de Fornecedores
- Use os filtros de período e fornecedor
- Analise métricas de qualidade e pontualidade
- Compare performance entre fornecedores

### 4. Log de Comunicação
- Registre todas as comunicações com fornecedores
- Use filtros para encontrar comunicações específicas
- Acompanhe follow-ups e pendências
