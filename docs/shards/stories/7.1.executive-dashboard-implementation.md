## 🎯 OBJETIVO

Implementar um dashboard executivo completo com KPIs principais, métricas de performance em tempo real, comparações período a período, alertas de performance, visualizações interativas e capacidades de exportação de relatórios.

## 📋 CRITÉRIOS DE ACEITAÇÃO

### 1. Dashboard Executivo com KPIs Principais
- [ ] **KPIs Financeiros**: Receita total, margem de lucro, ticket médio, crescimento mensal
- [ ] **KPIs Operacionais**: Taxa de ocupação, produtividade por profissional, tempo médio de atendimento
- [ ] **KPIs de Pacientes**: Novos pacientes, retenção, satisfação, no-shows
- [ ] **KPIs de Staff**: Utilização, produtividade, satisfação da equipe
- [ ] **Widgets configuráveis** com diferentes tipos de visualização

### 2. Métricas de Performance em Tempo Real
- [ ] **Atualização automática** dos dados a cada 5 minutos
- [ ] **Indicadores de status** (verde/amarelo/vermelho) baseados em thresholds
- [ ] **Notificações push** para métricas críticas
- [ ] **Histórico de performance** com tendências
- [ ] **Comparação com metas** estabelecidas

### 3. Comparações Período a Período
- [ ] **Comparação mensal** (mês atual vs anterior)
- [ ] **Comparação anual** (ano atual vs anterior)
- [ ] **Comparação semanal** para métricas operacionais
- [ ] **Análise de sazonalidade** e tendências
- [ ] **Variação percentual** e absoluta

### 4. Alertas de Performance
- [ ] **Sistema de alertas configurável** por métrica
- [ ] **Thresholds personalizáveis** por usuário/role
- [ ] **Notificações por email/SMS** para alertas críticos
- [ ] **Dashboard de alertas** com histórico
- [ ] **Escalação automática** para gestores

### 5. Visualizações Interativas
- [ ] **Gráficos drill-down** para análise detalhada
- [ ] **Filtros dinâmicos** por período, profissional, serviço
- [ ] **Mapas de calor** para identificar padrões
- [ ] **Gráficos de tendência** com projeções
- [ ] **Dashboards responsivos** para mobile

### 6. Exportação de Relatórios
- [ ] **Exportação em PDF** com layout executivo
- [ ] **Exportação em Excel** com dados detalhados
- [ ] **Agendamento automático** de relatórios
- [ ] **Compartilhamento por email** com stakeholders
- [ ] **Templates personalizáveis** de relatórios

## 🔧 REQUISITOS TÉCNICOS

### Database Schema Extensions
```sql
-- Executive Dashboard Tables
CREATE TABLE executive_dashboards (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  clinic_id UUID REFERENCES clinics(id),
  name VARCHAR(255) NOT NULL,
  layout JSONB NOT NULL, -- Grid layout configuration
  widgets JSONB NOT NULL, -- Widget configurations
  filters JSONB DEFAULT '{}',
  is_default BOOLEAN DEFAULT false,
  created_by UUID REFERENCES auth.users(id),
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

CREATE TABLE dashboard_widgets (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  dashboard_id UUID REFERENCES executive_dashboards(id),
  widget_type VARCHAR(100) NOT NULL, -- 'kpi', 'chart', 'table', 'metric'
  title VARCHAR(255) NOT NULL,
  config JSONB NOT NULL, -- Widget-specific configuration
  position JSONB NOT NULL, -- Grid position and size
  data_source VARCHAR(255) NOT NULL,
  refresh_interval INTEGER DEFAULT 300, -- seconds
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

CREATE TABLE kpi_definitions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  clinic_id UUID REFERENCES clinics(id),
  name VARCHAR(255) NOT NULL,
  category VARCHAR(100) NOT NULL, -- 'financial', 'operational', 'patient', 'staff'
  calculation_method TEXT NOT NULL, -- SQL or calculation logic
  target_value DECIMAL(15,2),
  warning_threshold DECIMAL(15,2),
  critical_threshold DECIMAL(15,2),
  unit VARCHAR(50), -- '%', 'R$', 'count', etc.
  is_active BOOLEAN DEFAULT true,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

CREATE TABLE kpi_values (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  kpi_id UUID REFERENCES kpi_definitions(id),
  clinic_id UUID REFERENCES clinics(id),
  period_start TIMESTAMP WITH TIME ZONE NOT NULL,
  period_end TIMESTAMP WITH TIME ZONE NOT NULL,
  value DECIMAL(15,2) NOT NULL,
  previous_value DECIMAL(15,2),
  target_value DECIMAL(15,2),
  status VARCHAR(20) DEFAULT 'normal', -- 'normal', 'warning', 'critical'
  calculated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  
  UNIQUE(kpi_id, period_start, period_end)
);

CREATE TABLE dashboard_alerts (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  clinic_id UUID REFERENCES clinics(id),
  kpi_id UUID REFERENCES kpi_definitions(id),
  alert_type VARCHAR(50) NOT NULL, -- 'threshold', 'trend', 'anomaly'
  severity VARCHAR(20) NOT NULL, -- 'info', 'warning', 'critical'
  title VARCHAR(255) NOT NULL,
  message TEXT NOT NULL,
  current_value DECIMAL(15,2),
  threshold_value DECIMAL(15,2),
  is_acknowledged BOOLEAN DEFAULT false,
  acknowledged_by UUID REFERENCES auth.users(id),
  acknowledged_at TIMESTAMP WITH TIME ZONE,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

CREATE TABLE report_templates (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  clinic_id UUID REFERENCES clinics(id),
  name VARCHAR(255) NOT NULL,
  description TEXT,
  template_type VARCHAR(50) NOT NULL, -- 'executive', 'operational', 'financial'
  layout JSONB NOT NULL,
  schedule JSONB, -- Cron-like schedule for automatic generation
  recipients JSONB, -- Email recipients
  is_active BOOLEAN DEFAULT true,
  created_by UUID REFERENCES auth.users(id),
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

CREATE TABLE generated_reports (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  template_id UUID REFERENCES report_templates(id),
  clinic_id UUID REFERENCES clinics(id),
  period_start TIMESTAMP WITH TIME ZONE NOT NULL,
  period_end TIMESTAMP WITH TIME ZONE NOT NULL,
  file_path VARCHAR(500),
  file_size INTEGER,
  generation_status VARCHAR(50) DEFAULT 'pending', -- 'pending', 'generating', 'completed', 'failed'
  generated_by UUID REFERENCES auth.users(id),
  generated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  sent_at TIMESTAMP WITH TIME ZONE
);
```
