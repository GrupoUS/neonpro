## 🛠️ Technical Architecture

### 🎨 Frontend Stack
- **Framework**: Next.js 15 with App Router
- **UI Library**: React 19 with shadcn/ui components
- **Language**: TypeScript for type safety
- **Styling**: Tailwind CSS with custom healthcare themes
- **State Management**: Zustand with persistence and hydration
- **Testing**: Jest + React Testing Library with healthcare utilities

### 🔧 Backend & Database
- **Database**: Supabase (PostgreSQL) with Row Level Security
- **Authentication**: Supabase Auth with MFA integration
- **API**: Next.js API routes with middleware protection
- **File Storage**: Supabase Storage for documents and images
- **Real-time**: Supabase Realtime for live updates
- **Migrations**: Comprehensive database schema with audit functions

### 🚀 DevOps & Deployment
- **Package Manager**: pnpm with workspace optimization
- **Build Tool**: Turbo for monorepo builds
- **Code Quality**: Biome for formatting and linting
- **Git Hooks**: Husky + lint-staged for pre-commit validation
- **CI/CD**: GitHub Actions with security scanning
- **Deployment**: Vercel with automatic deployments
- **Monitoring**: Application and performance monitoring
